# 模式选择

## 模式 A：识别后再试
当用户还没有完全锁定方程，只明确了大致研究问题时，先做以下判断：

1. `x` 和 `y` 是否已被老师、文献或既有研究设计明确？
2. 当前有没有可继承的基准模型？
3. 现在更不稳定的是解释变量、被解释变量，还是两者兼有？

若任一关键项缺失，不要直接进入大规模规格搜索。

### 模式 A 的最少问题

- 你现在明确的 `x` 和 `y` 分别是什么？
- 这个关系是老师定的、文献借鉴的，还是你自己拟定的？
- 你希望优先试 `x`、试 `y`，还是都试？
- 你目前认可的基准估计方法是什么？

## 模式 B：已有方程，直接推进
当用户已经给出以下任一内容时，默认进入模式 B：

- 现成方程
- 现成 do 文件
- 一轮回归结果
- 明确要求“按这个模型继续试”

### 模式 B 的默认动作

1. 读取现有规格
2. 判断是试 `x` 还是试 `y`
3. 在当前模型附近优先做变换
4. 失败后再进入替代指标思路

## 决策规则

- 有现成模型时，默认模式 B
- 只有研究问题、没有方程时，默认模式 A
- 用户同时给了研究问题和现成结果时，优先模式 B
